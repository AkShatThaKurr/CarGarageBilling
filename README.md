# 🚗 Garage Billing System

A console-based Java application to manage a car service center.

## 📌 Features
- Add customer details
- Store car information
- Select services
- Generate invoice

## 🛠️ Technologies Used
- Java
- OOP (Object-Oriented Programming)

## ▶️ How to Run
1. Compile all Java files
2. Run `GarageBillingApp.java`
3. Follow menu instructions

## 📈 Future Improvements
- Add database using JDBC
- Integrate MySQL
- Convert to Spring Boot REST API
